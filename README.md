# 📘 AskMyDoc
A Retrieval-Augmented Generation (RAG) powered document Q&A app built with **LangChain** and **Streamlit**.  
Upload your `.pdf`, `.txt`, or `.md` files and ask questions — get accurate, context-aware answers instantly.
